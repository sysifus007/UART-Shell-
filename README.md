# UART-Shell

> STM32 裸机串口命令行交互系统 -- 类 Linux Shell 体验

[![Language](https://img.shields.io/badge/Language-C-blue)]()
[![MCU](https://img.shields.io/badge/MCU-STM32F103-blue)]()

## 项目简介

基于 STM32F103 实现类 Linux 风格的串口命令行交互界面。采用分层架构设计，核心框架与硬件完全解耦，支持命令注册分发、Tab 自动补全、命令历史记录等高级功能。

## 功能特性

| 功能 | 说明 |
|------|------|
| 命令注册分发 | 支持命令动态注册/注销，可扩展性强 |
| Tab 自动补全 | 按 Tab 键自动补全命令名 |
| 命令历史记录 | 上下键浏览历史命令 |
| ANSI 彩色输出 | 支持终端颜色高亮显示 |
| 参数解析 | 支持引号包裹参数、多参数解析 |
| 内置 16 个命令 | 覆盖 LED 控制、传感器读取、系统信息等场景 |

## 架构设计

```
+-------------------+
|  Application Layer |   <-- 16 个业务命令 (led, sensor, sysinfo...)
+-------------------+
|   Shell Framework  |   <-- 命令注册/分发/补全/历史/解析
+-------------------+
|    HAL Adapter     |   <-- UART 发送/接收硬件抽象
+-------------------+
|    Hardware        |   <-- STM32F103 USART
+-------------------+
```

核心框架层（Shell Framework）与硬件完全解耦，移植到其他 MCU 只需适配 HAL Adapter 层。

## 内置命令

| 命令 | 功能 |
|------|------|
| help | 显示所有可用命令 |
| led on/off | 控制 LED 开关 |
| sensor read | 读取传感器数据 |
| sysinfo | 显示系统信息 |
| history | 查看命令历史 |
| clear | 清屏 |
| ... | 更多命令请查看 help |

## 技术亮点

1. **分层架构**: 应用层 / 框架层 / 适配层三层分离，框架零硬件依赖
2. **Tab 补全**: 前缀匹配 + 通用前缀提取，体验接近 Linux Bash
3. **ANSI 支持**: 终端颜色高亮，提升可读性
4. **引号解析**: 支持 `"hello world"` 作为单个参数
5. **动态注册**: 运行时注册/注销命令，无需修改框架代码
