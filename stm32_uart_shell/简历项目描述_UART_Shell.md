## 精简版（适合简历空间有限）

**串口命令行终端（UART Shell/CLI）** | STM32F103 + C 语言 | 个人项目

基于 STM32 单片机实现类 Linux 风格的串口命令行交互界面。采用**分层架构**设计（应用命令层 / Shell 核心框架 / HAL 适配层），核心框架与硬件完全解耦。实现了命令注册分发、Tab 自动补全、命令历史记录（上下键浏览）、ANSI 彩色输出、参数解析（支持引号包裹）等功能。内置 16 个命令覆盖 LED 控制、传感器读取、系统信息等场景，支持命令动态注册/注销，可扩展性强。

## 完整版（推荐用于详细项目列表）

**串口命令行终端（UART Shell/CLI）**

**技术栈：** STM32F103C8T6 / C 语言 / STM32 HAL 库 / USART / GPIO / ADC

**项目描述：**
在 STM32F103 单片机上从零设计并实现了一个轻量级的串口命令行交互框架，通过电脑串口终端（115200-8-N-1）输入指令即可实时控制硬件外设、查询系统状态。该框架采用**三层架构**，硬件无关的核心代码可直接移植到其他 MCU 平台。

**技术亮点：**

- **分层架构设计：** 应用命令层（cmd_system / cmd_led / cmd_sensor）+ Shell 核心层（解析器 / 注册表 / 行编辑器）+ HAL 适配层，各层职责清晰，通过 6 个函数指针完成硬件解耦
- **命令注册机制：** 基于函数指针表的动态命令注册/注销系统，新增命令只需定义 `shell_cmd_t` 结构体并调用 `shell_register()`，无需修改核心代码
- **行编辑功能：** 支持 Backspace 删除、光标左右移动、Tab 自动补全（显示候选列表 + 公共前缀补全）、上下键浏览历史记录（环形缓冲区存储 16 条）
- **ANSI 终端渲染：** 通过 VT100 转义码实现彩色输出、光标控制、清屏等效果，提升调试体验
- **参数解析器：** 支持 argc/argv 风格的参数分割，兼容引号包裹的空格、`#` 注释行跳过
- **中断驱动 + DMA 可选：** UART 接收支持逐字节中断模式和 DMA 空闲中断模式，中断上下文中完成行编辑，主循环空闲可执行其他任务

**内置命令（16 个）：**

| 模块 | 命令 | 功能 |
|------|------|------|
| 系统 | `help` / `version` / `reboot` / `uptime` / `free` / `ps` / `date` / `echo` / `clear` / `history` | 固件信息、系统复位、运行时间、内存概览 |
| LED | `led on/off/toggle/status/blink` | 板载 LED 控制（支持闪烁次数和间隔） |
| 传感器 | `read_temp` / `read_adc` / `read_gpio` / `read_dht` | 温度、ADC 多通道采集、GPIO 电平、温湿度 |

**项目成果：**
- 核心 Shell 框架代码约 500 行，零外部依赖，RAM 开销 < 2KB
- 提供了 PC 模拟版（替换 HAL 层为 stdin/stdout），脱离硬件即可验证全部功能
- 设计了统一的 HAL 适配接口，更换 MCU 型号仅需重写 6 个函数

## 面试可能被问到的问题（准备参考）

1. **为什么不用现成的库（如 MicroShell、CLI library）？**
   > 理解原理更重要。自己实现可以深入理解命令解析、状态机、环形缓冲区、UART 中断等底层机制。

2. **Shell 在中断中处理是否安全？**
   > `shell_feed()` 只做缓冲区操作和单字节输出，无阻塞调用和动态内存分配。命令执行（handler）在收到 `\r` 后触发，可移至主循环中执行以进一步降低中断占用。

3. **如何扩展一个新命令？**
   > 三步：(1) 编写 `int cmd_xxx(int argc, char *argv[])` 函数；(2) 定义 `shell_cmd_t` 结构体；(3) 调用 `shell_register()`。核心代码不需要任何修改。

4. **DMA 模式和中断模式有什么区别？**
   > 中断模式每收到一个字节触发一次 ISR，适合低波特率；DMA + IDLE 模式在空闲时一次性接收整帧，CPU 开销更低，适合高波特率或大数据量场景。

5. **ANSI 转义码在某些串口助手不显示颜色怎么办？**
   > 通过 `SHELL_USE_ANSI` 宏控制开关，设为 0 时所有颜色码为空字符串，自动退化为纯文本输出。
